# 🎮 Змейка на Swing

## Описание
Графическая версия игры "Змейка" с использованием Java Swing.

## Особенности
- 🎨 Красивый интерфейс с градиентами
- 🎯 4 уровня сложности
- 🎨 Настройка цвета змейки
- 💾 Сохранение рекордов
- ⏸️ Пауза и возобновление
- 📊 Система уровней

## Компиляция и запуск
```bash
javac com/snake/*.java
java com.snake.SnakeGame
