# 🚀 Продвинутая Змейка на JavaFX

## 🌟 Особенности

### 🎮 Геймплей
- **4 уровня сложности**: Easy, Medium, Hard, Expert
- **5 типов еды** с уникальными эффектами
- **Power-ups**: ускорение, неуязвимость, замедление
- **Процедурная генерация стен**
- **Система уровней** (1-10+)

### 🎨 Графика
- JavaFX с анимациями
- Градиенты и спецэффекты
- Анимированные power-ups
- Частицы при поедании еды

### 🔊 Звук
- Фоновая музыка
- Звуковые эффекты
- Регулировка громкости

### 📊 Прогресс
- SQLite база данных
- Таблица рекордов
- Система достижений
- Статистика игрока

## 🎯 Типы еды

| Тип | Цвет | Очки | Эффект |
|------|------|------|---------|
| Обычная | 🔴 Красный | 10 | - |
| Бонусная | ⭐ Золотой | 50 | Появляется power-up |
| Скорость | ⚡ Голубой | 20 | Ускорение на 5с |
| Замедление | 🐢 Фиолетовый | 20 | Замедление на 3с |
| Золотая | 👑 Оранжевый | 100 | Неуязвимость на 3с |

## 🎮 Управление

- **Стрелки** или **WASD** - движение
- **Пробел** - пауза/рестарт
- **ESC** - меню

## 📦 Установка и запуск

### Требования
- Java 11 или выше
- Maven 3.6+

### Запуск
```bash
# Клонирование
git clone https://github.com/yourusername/snake-10-languages.git
cd snake-10-languages/java/03_advanced

# Сборка
mvn clean compile

# Запуск
mvn javafx:run

# Или сборка JAR
mvn package
java -jar target/snake-advanced-1.0.0.jar
