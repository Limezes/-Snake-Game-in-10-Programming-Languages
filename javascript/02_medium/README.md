# 🎮 Змейка на Canvas

## Описание
Графическая версия игры "Змейка" с использованием HTML5 Canvas.

## Особенности
- 🎨 Красивая графика с градиентами
- 📱 Адаптивный дизайн
- 🎯 Три уровня сложности
- 💾 Сохранение рекорда в localStorage
- ⏸️ Пауза и возобновление игры

## Управление
- **Стрелки** или **WASD** - движение
- **Пробел** - пауза/продолжение
- **Кнопки** на экране для управления

## Уровни сложности
- **Легкий**: медленная скорость (150мс)
- **Средний**: средняя скорость (100мс)
- **Сложный**: высокая скорость (70мс)

## Запуск
Просто откройте `index.html` в любом современном браузере!
