# 🚀 Продвинутая Змейка на React + TypeScript

## 🌟 Особенности

### 🎮 Геймплей
- **4 уровня сложности**: Easy, Medium, Hard, Expert
- **5 типов еды** с уникальными эффектами
- **Система уровней** с процедурной генерацией стен
- **Power-ups**: ускорение, замедление, неуязвимость
- **Анимации** с Framer Motion

### 🎨 Графика
- Стильный темный интерфейс
- Плавные анимации
- Эффекты свечения и частиц
- Адаптивный дизайн

### 📊 Прогресс
- Сохранение рекордов в localStorage
- Система уровней (1-4+)
- Статистика игры
- Достижения (скоро)

### 🔧 Технологии
- React 18
- TypeScript
- Framer Motion для анимаций
- Styled Components
- React Hot Toast для уведомлений
- Socket.IO для мультиплеера (готовится)

## 🎯 Типы еды

| Тип | Цвет | Очки | Эффект |
|------|------|------|---------|
| Обычная | 🔴 Красный | 10 | - |
| Бонусная | ⭐ Желтый | 50 | Доп. очки |
| Скорость | ⚡ Синий | 20 | Ускорение на 10с |
| Замедление | 🐢 Фиолетовый | 20 | Замедление на 10с |
| Золотая | 👑 Оранжевый | 100 | Неуязвимость на 15с |

## 🎮 Управление

- **Стрелки** или **WASD** - движение
- **Пробел** - пауза
- **ESC** - выход в меню

## 📦 Установка и запуск

```bash
# Клонирование
git clone https://github.com/yourusername/snake-10-languages.git
cd snake-10-languages/javascript/03_advanced

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка для продакшна
npm run build
