# 🐍 Консольная змейка на C#

## Описание
Простая версия игры "Змейка" для консоли на C#.

## Компиляция и запуск

### С помощью .NET CLI
```bash
dotnet new console -n SnakeGame
cd SnakeGame
# Замените Program.cs нашим кодом
dotnet run
```

### С помощью Visual Studio
1. Создайте новый проект "Консольное приложение"
2. Замените Program.cs нашим кодом
3. Нажмите F5 для запуска

## Управление
- **WASD** или **Стрелки** - движение змейки
- **Q** - выход из игры
- **R** - рестарт после окончания игры

## Особенности
- Чистый C# без внешних зависимостей
- Unicode символы для графики
- Плавное управление
- Автоматическая очистка консоли
