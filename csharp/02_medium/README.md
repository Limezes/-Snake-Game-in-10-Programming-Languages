# 🎮 Змейка на WinForms

## Описание
Графическая версия игры "Змейка" с использованием Windows Forms.

## Особенности
- 🎨 Красивый интерфейс
- 🎯 Три уровня сложности
- 💾 Сохранение рекордов
- ⏸️ Пауза и возобновление
- 🎨 Разные типы еды

## Управление
- **Стрелки** или **WASD** - движение
- **Пробел** - пауза/продолжение

## Компиляция и запуск

### Visual Studio 2022
1. Откройте проект `SnakeGame.csproj`
2. Нажмите F5 для запуска

### .NET CLI
```bash
dotnet build
dotnet run
```

## Системные требования
- .NET 6.0 или выше
- Windows (для WinForms)
