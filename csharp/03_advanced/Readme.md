# 🚀 Продвинутая Змейка на WPF + MVVM

## 🌟 Особенности

### 🎮 Геймплей
- **4 уровня сложности**: Easy, Medium, Hard, Expert
- **5 типов еды** с уникальными эффектами
- **5 типов power-ups** (ускорение, неуязвимость, замедление, доп. жизнь, множитель)
- **Процедурная генерация стен**
- **Система уровней** с возрастающей сложностью

### 🎨 Архитектура
- **MVVM паттерн** для чистой архитектуры
- **Dependency Injection** через Microsoft.Extensions.DependencyInjection
- **Команды** для обработки действий пользователя
- **Конвертеры** для XAML биндингов

### 📊 Данные
- **JSON сериализация** для настроек
- **SQLite база данных** для рекордов и статистики
- **Рейтинг игроков** и достижения

### 🔊 Аудио
- **SoundPlayer** для звуковых эффектов
- **MediaPlayer** для фоновой музыки
- Регулировка громкости

## 🎯 Типы еды

| Тип | Цвет | Очки | Эффект |
|------|------|------|---------|
| Обычная | 🔴 Красный | 10 | - |
| Бонусная | ⭐ Золотой | 50 | Мгновенный бонус |
| Скорость | ⚡ Голубой | 20 | Ускорение на 5с |
| Замедление | 🐢 Фиолетовый | 20 | Замедление на 5с |
| Золотая | 👑 Оранжевый | 100 | Неуязвимость на 8с |

## ⚡ Power-ups

| Тип | Цвет | Эффект | Длительность |
|------|------|---------|---------------|
| Ускорение | 🔵 Голубой | x2 скорость | 5с |
| Неуязвимость | 🟡 Желтый | Без урона | 5с |
| Замедление | 🟣 Фиолетовый | x0.5 скорость | 5с |
| Доп. жизнь | 🔴 Красный | +1 жизнь | Мгновенно |
| Множитель | 🟠 Оранжевый | x2 очков | Мгновенно |

## 🎮 Управление

- **Стрелки** или **WASD** - движение
- **Пробел** - пауза/продолжение
- **ESC** - меню
- **Ctrl+N** - новая игра

## 📦 Установка и запуск

### Требования
- .NET 6.0 SDK или выше
- Visual Studio 2022 (рекомендуется) или VS Code

### Запуск
```bash
# Клонирование
git clone https://github.com/yourusername/snake-10-languages.git
cd snake-10-languages/csharp/03_advanced/SnakeGame

# Восстановление пакетов
dotnet restore

# Сборка
dotnet build

# Запуск
dotnet run
```

### В Visual Studio
1. Откройте `SnakeGame.sln`
2. Нажмите F5 для запуска

## 🔧 Настройки

- **Сложность** - регулирует скорость игры и наличие стен
- **Имя игрока** - для таблицы рекордов
- **Звук** - включение/выключение звуковых эффектов
- **Громкость** - регулировка громкости (0-100)
- **Сетка** - отображение сетки на игровом поле

## 🏆 Достижения

- 🥇 **Первая еда** - Съесть первое яблоко
- 💯 **100 очков** - Набрать 100 очков
- 🏆 **500 очков** - Набрать 500 очков
- 👑 **1000 очков** - Набрать 1000 очков
- 📈 **5 уровень** - Достичь 5 уровня
- 📊 **10 уровень** - Достичь 10 уровня
- ⚡ **Демон скорости** - Достичь максимальной скорости
- 🛡️ **Неуязвимый** - Съесть 10 бонусов подряд
- 🧱 **Разрушитель стен** - Пройти уровень со стенами
- 👑 **Золотой пир** - Съесть 5 золотых яблок

## 📊 Статистика

- Общее количество игр

- Средний счет

- Собрано power-ups
