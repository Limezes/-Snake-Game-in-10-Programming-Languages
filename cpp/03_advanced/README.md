# 🚀 Продвинутая Змейка на Qt

## 🌟 Особенности

### 🎮 Геймплей
- **4 уровня сложности**: Easy, Medium, Hard, Expert
- **5 типов еды** с уникальными эффектами
- **5 типов power-ups** (ускорение, неуязвимость, замедление, доп. жизнь, множитель очков)
- **Процедурная генерация стен**
- **Система уровней** с возрастающей сложностью

### 🎨 Графика
- Qt Widgets с кастомной отрисовкой
- Плавные анимации
- Эффекты свечения и пульсации
- Настраиваемые цвета

### 🔊 Аудио
- Звуковые эффекты (поедание, level-up, game over)
- Фоновая музыка
- Регулировка громкости

### 📊 База данных (SQLite)
- Сохранение рекордов
- Система достижений
- Детальная статистика
- Рейтинг игроков

## 🎯 Типы еды

| Тип | Цвет | Очки | Эффект |
|------|------|------|---------|
| Обычная | 🔴 Красный | 10 | - |
| Бонусная | ⭐ Золотой | 50 | Мгновенный бонус |
| Скорость | ⚡ Голубой | 20 | Ускорение на 5с |
| Замедление | 🐢 Фиолетовый | 20 | Замедление на 5с |
| Золотая | 👑 Золотой | 100 | Неуязвимость на 8с |

## ⚡ Power-ups

| Тип | Цвет | Эффект | Длительность |
|------|------|---------|---------------|
| Ускорение | 🔵 Голубой | x2 скорость | 8с |
| Неуязвимость | 🟡 Желтый | Без урона | 8с |
| Замедление | 🟣 Фиолетовый | x0.3 скорость | 5с |
| Доп. жизнь | 🔴 Красный | +1 жизнь | Мгновенно |
| Множитель | 🟠 Оранжевый | x2 очков | Мгновенно |

## 🎮 Управление

- **Стрелки** или **WASD** - движение
- **Пробел** - пауза/продолжение
- **ESC** - меню
- **Ctrl+N** - новая игра
- **Ctrl+Q** - выход

## 📦 Установка и запуск

### Требования
- Qt 5.15 или выше
- Компилятор с поддержкой C++11
- SQLite (встроен в Qt)

### Сборка
```bash
# Клонирование
git clone https://github.com/yourusername/snake-10-languages.git
cd snake-10-languages/cpp/03_advanced

# Сборка с qmake
qmake snake.pro
make

# Запуск
./SnakeAdvanced
